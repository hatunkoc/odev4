JavaScript’e Giriş

Bu yazımızda javascript ile ilgili temel bilgileri paylaşacağım. Javascript Netscape firması tarafından C tabanında geliştirilmiş bir script dilidir. Script dillerinin özelliği derlenebilir bir dosya oluşturmaz. JS uzantılı dosyalarla sunucu üzerinden çalışır. Javascript web sayfalarında client tarafından çalıştığı için hızlıdır.

JavaScript’in özellikleri

  • Kullanıcı taraflı çalışan bir dildir. (client)
  • Javascript kodlarıyla bilgisayardan herhangi bir dosya okunamaz,yazılamaz ve dosya oluşturulamaz. Çerezler bu durumun dışındadır.
  • Html sayfaların bir parçasıdır. Tek başlarına bir anlam ifade etmezler.
  • Tarayıcılar tarafından yorumlanır. Bu yine kullanıcı taraflılığıyla ilgilidir.
  • Html içine gömüldüğü gibi .js uzantılı ek sayfa olarakta kullanılabilir.
  • Herhangi bir metin editörüyle geliştirilebilir.
  • Tarayıcılarda farklılık gösterebilir. İE bu konuda çok başarılıdır(!). Neyseki bunu jQuery kütüphaneleriyle çözüme kavuşturduk. İleride belki jQueryede sitemizde yer verebiliriz.

Yazım kuralları

  • Javascriptte bütün değişkenler Harf yada (_) altçizgi ile başlar. Rakam ile başlayamaz.
  • Değişken isimleri 255 karakter uzunluğundan fazla olamaz.
  • Javascript’te büyük küçük harf duyarlılığı vardır. Deger isimli değişeken ile deger isimli değişken farklı algılanır.
  • Türkçe karakter kullanılamaz.(ı,ş,ğ,ü,ö,ç)
  • Değişkenler var ifadesiyle tanımlanır.
  • Sabit bir değişkeni const SABİT_DEGER şeklinde tanımlamalısınız. Const yazımı yanlıştır.(madde 3).
  • İfade tanımlarında anahtar kelimeler kullanılamaz. (if, for, while vs.)
  • Açılan süslü parantezlerin kapatılması gerekmektedir. ( { } )
  • Javascriptte ‘(tek tırnak) ile “(çift tırnak) işareti arasında işlevsel bakımdan fark yoktur. Ancak iç içe kullanımda birbiriyle çakışmamasına özen gösterilmelidir.
